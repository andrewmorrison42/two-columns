# Two Columns — setup

Same deployment as Daily Bread. If you've done that once, this is familiar.

## What it is
A single-purpose tool for the moment you're hooked on an interpersonal
situation — bracing for someone's reaction, replaying a conversation, tempted
to send a softening follow-up. It walks one path:

1. **Name it** in one line.
2. **Step back** — an unhook exercise (leaves on a stream, sky not the weather,
   or a card on the lap). You can't skip past it; the pause is enforced.
3. **Sort** — put each thought in one of two columns: *the integrity of my
   action* or *their reaction*.
4. **Check** — the hinge: was your action values-aligned? It won't help you with
   their reaction until you've answered this.
5. **Act or repair** — if integrity is settled, the only live question is your
   next values-based move; if it isn't, the work is to make the action right,
   not to manage the reaction.

It saves **nothing**. No journal, no streak, no history. It does its work in the
moment and lets go — that restraint is deliberate, so it can't become another
system to tend or a place to hide.

## Put it online (free)
Same as before. Easiest: a new GitHub repo (e.g. `two-columns`), upload the
seven files at the top level, turn on **Settings → Pages → Deploy from a branch
→ main / root**. Or drag the files onto app.netlify.com/drop for an instant link.

Keep it in its **own** repo, separate from daily-bread, so each has its own
address.

## Install on your phone
- **iPhone (Safari):** open the link → Share → Add to Home Screen.
- **Android (Chrome):** open the link → ⋮ menu → Add to Home screen / Install.

First load needs internet; after that it works offline.

## A note on the icon
A single leaf resting on still water, not an alarm button. The app is a pause,
not an emergency brake — the icon is meant to read that way every time you see it.
